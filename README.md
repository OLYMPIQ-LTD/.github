# Olympiq LTD

**Olympiq LTD** develops innovative software for **sports clubs**. Our flagship product, **Olympiq**, is an all-in-one **membership management and session booking platform** designed for clubs, coaches, and administrators.

Unlike competitors like **Spond**, which only handle session bookings, Olympiq provides **full membership management**, including:

- Account login and management  
- Payment processing via Stripe (no card data stored)  
- Age verification and parental consent for minors  
- Emergency contact tracking  
- Optional push notifications and newsletters  

In addition to the app, clubs get a **customizable, professional-looking website** to showcase their club, advertise sessions, and engage members online.
